## Descriptive Title

Allow NFC Scanning to Allow Reading vCards (.vcf files)

## Request Type

Interoperability Request

## Feature Name

NameDrop (NFC), Background NFC Tag Reading

## Reason for Request

I would like to be able to scan an NFC tag in iOS and read a contact card (.vcf file). This is
currently supported by Android but not by Apple unless using the NameDrop feature. This is a
huge limitation that is not a technical limitation of iOS.

## Product Description

Physical NFC tag

## How the Product Will Use the Feature

NFC tag read by iOS device will allow importing .vcf file as contact.

## Where the Product Is/Will Be Offered

United States
European Union

## Other Frameworks Evaluated

The only way to use Background NFC Tag Reading (as far as I'm aware) is through what is built
into iOS. Requiring using a third party iOS app is unacceptable as it prevents contact sharing
for iOS users that don't have a third party app installed.
